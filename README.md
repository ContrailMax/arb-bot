## Setup
1) Create Google Service Account + share the sheet to service account email.
2) Add GitHub Secret: GDRIVE_API_KEY (full JSON).
3) In Google Sheet tab "Setting", set:
   - ALERT_BPS, COOLDOWN_MIN, COINS, TELEGRAM_BOT_TOKEN, TELEGRAM_CHAT_ID
4) Run workflow manually or wait for schedule (every 5 minutes).
